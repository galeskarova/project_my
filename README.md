```print(mask_account_card("Visa Platinum 1234567890123456"))```
Вывод: Visa Platinum 1234 56** **** 3456

### Преобразование даты
```print(get_date("2024-03-11T12:30:45.123456"))```

## Документация и ссылки
Описание функций
get_mask_card_number(card_number: str) -> str
Принимает номер карты (строкой или числом) и возвращает маску в формате XXXX XX** **** XXXX.

get_mask_account(account: str) -> str
Принимает номер счета, возвращает маску вида **XXXX (только последние 4 цифры).

mask_account_card(account_card: str) -> str
Анализирует строку, содержащую тип и номер карты/счета. Если строка начинается с "Счет", применяет маскировку счета, иначе — карты. Возвращает строку с замаскированным номером.

get_date(date: str) -> str
Преобразует строку с датой в формате "%Y-%m-%dT%H:%M:%S.%f" в формат "ДД.ММ.ГГГГ".

filter_by_state(list_of_dicts: list, state: str = 'EXECUTED') -> list
Фильтрует список словарей, оставляя только те, у которых значение ключа 'state' равно заданному. По умолчанию state='EXECUTED'.

sort_by_date(list_of_dicts: list, reverse: bool = True) -> list
Сортирует список словарей по ключу 'date'. Параметр reverse=True (по умолчанию) означает сортировку по убыванию (сначала новые даты), False — по возрастанию.
